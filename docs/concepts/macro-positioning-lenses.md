# 宏观镜头 + 期权持仓镜头(Macro & Positioning Lenses)

> 个人知识笔记。解释 2026-06 加入的两个"镜头(lens)"以及里面用到的术语。
> 内容较多,可按一级标题(`##`)拆成多个小文件放进本文件夹。术语集中在文末「术语表」。

---

## 1. 总体设计:为什么放在「边缘层」

整个系统分两层:

- **引擎层**(`quant/scoring.py`、`quant/decision.py`):纯 Python、确定性、**可回测**。
  给每只股票打分,给出「加仓 / 减仓 / 持有 / 清仓」等机械意图。
- **边缘层**(`.claude/skills/` 里的 skill,也就是 Claude 读报告后做的判断):负责加「人/AI 的判断」。

这次加的两个镜头都放在**边缘层**:它们只是**给报告补充背景信息**,
**绝不参与打分和决策**——所以回测结果完全不变。

> 这点很关键:它保证引擎依然干净、可验证;同时新加的数字会**存进每日 parquet 数据库**
> (`data/daily_observations/<profile>/`),为以后「用真实涨跌来给历史判断打分」(评估/反思闭环)留好接口。

和这一层同性质的老模块:`valuation.py`(估值)、`roles.py`(角色)、`option_flow.py`(期权墙)——
都是「只读背景」,从不喂给引擎。

## 2. Part A:宏观镜头(Macro lens)

### 它解决什么问题
原来的引擎判断「大盘环境」只看三样:SPY、QQQ、**VIX**(恐慌指数)。太窄。
**宏观环境**(利率、信用、流动性)对长久期的 AI/科技股影响巨大,但引擎完全看不到。Part A 把这块补上。

### 数据从哪来:FRED
- **FRED** = Federal Reserve Economic Data,美联储(圣路易斯分行)的免费经济数据库。
- 用的是它**不需要 API key** 的 CSV 下载接口,所以零配置、不花钱。
- 代码:`quant/providers.py` 的 `fetch_macro()`(每日缓存,失败回退旧缓存)。

### 「6 个 series」是哪 6 个
`series` = 一条数据序列 / 一个指标。配置在每个 profile 的 `config.yaml: macro.series`。

| 代码 | 中文含义 | 它告诉你什么 |
|---|---|---|
| **DGS10** | 10 年期美国国债收益率 | 长期「无风险利率」。它是给科技股未来现金流「打折」的折现率——利率越高,远期利润越不值钱,成长股估值越受压。 |
| **DGS2** | 2 年期国债收益率 | 对美联储短期政策最敏感。 |
| **DFII10** | 10 年期**实际**收益率(扣掉通胀预期) | **对科技股最关键的一条**。实际利率上升 = 「久期逆风(duration headwind)」,最长久期的成长股最受伤;下降 = 「久期顺风」。 |
| **T10YIE** | 10 年期盈亏平衡通胀率 | 市场预期的未来 10 年平均通胀,是判断美联储下一步的背景。 |
| **BAMLH0A0HYM2** | 高收益债信用利差(HY spread) | 垃圾债比国债多要的利息。**利差走阔 = 市场避险(risk-off)**,是股市下跌的早期预警;利差低且稳 = 风险偏好健康(risk-on)。 |
| **NFCI** | 芝加哥联储金融环境指数 | 综合的「钱松不松」指标。负数 = 宽松(loose),正数 = 收紧(tight)。 |

### 引擎给的「粗判断」
`quant/macro.py` 的 `detect_macro()` 把上面 6 个数,按阈值翻成几句标签(都可在 `config.yaml: macro.thresholds` 调):

- **利率方向**(10 年涨/跌/平)
- **收益率曲线**(2s10s:正常 / 平 / 倒挂)
- **久期顺/逆风**(看实际利率 DFII10 的方向)
- **信用**(risk-on / risk-off,看 HY 利差)
- **金融环境**(宽松 / 中性 / 收紧,看 NFCI)
- 再合成一句 `backdrop`,例如:`duration neutral, credit calm, conditions loose`(久期中性、信用平稳、环境宽松)

> 这只是**背景色**,不是交易信号。它**与 `MarketState` 并行**,不会被合并进大盘 regime,
> 所以不影响 `decision`,回测不变。

### 报告里会出现什么
weekly / daily 报告顶部多出一段 **"Macro backdrop"**;同时 daily 把
`dgs10 / real_yield / hy_spread / nfci / macro_backdrop` 这几列存进当日 parquet。

### 新 skill:`macro-review`
引擎只知道**数字**(比如实际利率涨了 8 个基点),但**不知道日历**——这就是 skill 的活:
用网络搜索补上**未来的事件**(FOMC/CPI/PCE/NFP),再结合你的科技股仓位,
给「顺风 / 逆风 / 中性」的判断,以及「什么会改变这个判断」的触发线。

## 3. Part B:期权持仓镜头扩展(Positioning)

你原来已经有「墙(walls)/ 最大痛点 / 预期波动 / IV 偏斜」。这次加了两样更定量的。
代码在 `quant/option_flow.py`。

### 3.1 庄家 Gamma / GEX(`gamma_flip`、`net_gex`)
背景:你买卖期权的对手方通常是做市商(**dealer**)。他们为了对冲风险,会**被迫**在股价
波动时买卖正股——这会反过来影响股价。

- **Gamma(伽马)**:期权的一个希腊字母,衡量「对冲量随股价变化的快慢」。
- **net_gex**(净 Gamma 敞口):把全期权链上每个行权价的 `gamma × 未平仓量` 加总后的净值。
  - **正数** = 做市商净多 gamma → 对冲行为**抑制**波动(股价倾向被「钉住 / 回归」)。
  - **负数** = 做市商净空 gamma → 对冲行为**放大**波动(下跌时越跌越卖,容易「踩空 / 瀑布式下跌」)。
- **gamma_flip(Gamma 翻转价)**:净 gamma 由正转负的那个股价。
  - 股价**在翻转价之上** → 通常处于「抑制波动」区:追涨胜率低、回调容易被买回。
  - 股价**在翻转价之下** → 「放大波动」区:下跌可能加速,**别接飞刀,要分批进**。

报告里会写成一句话,例如:
`spot above gamma flip $188 — dealers long-gamma, moves dampened`(现价在翻转价之上,波动被抑制)。

> ⚠️ 免费数据,未平仓量(OI)滞后约 1 个交易日,是**背景参考,不是触发信号**。
> 计算复用了 `quant/options.py` 的 `bs_gamma`(Black-Scholes 伽马)。

### 3.2 IV Rank(隐含波动率分位,`iv_rank`)
- **IV(Implied Volatility,隐含波动率)**:期权价格里「市场预期的未来波动」。IV 高 = 期权贵。
- **IV Rank**:今天的 IV 在**过去一段历史**里处于什么分位(0–100%)。
  - **高(贵)** → 适合**卖**期权收钱(例如在支撑位卖 cash-secured put 来接货)。
  - **低(便宜)** → 适合**买**期权来表达观点。
- 妙处:它直接用你每日库里**已经天天在存的 `atm_iv`** 来算——和你「积累数据、以后打分」的长期目标天然衔接。
- **现在显示「—」是正常的**:需要先积累约 **60 天**数据才会出数(配置 `option_positioning.iv_rank_min_days`),
  在那之前保持空值,不乱编。

### 出现在哪
- 报告 positioning 表新增两列:**Gamma flip**、**IV rank**。
- pretrade 简报的「重新锚定价位」里(到 gamma 翻转价的距离 + IV rank 一句话)。
- `pretrade-check` / `daily-review` 两个 skill 会主动帮你解读。

## 术语表(Glossary)

**数据/机构**
- **FRED** — 美联储(圣路易斯分行)免费经济数据库。
- **FOMC** — 联邦公开市场委员会,美联储**议息会议**(决定加不加息)。市场最看重的事件。
- **CPI** — 消费者物价指数,最常用的通胀数据。
- **PCE** — 个人消费支出物价指数,美联储**最看重**的通胀指标。
- **NFP** — 非农就业(每月第一个周五公布,看就业强弱)。
- **VIX** — 标普 500 隐含波动率指数,俗称「恐慌指数」。

**利率/宏观**
- **基点 (bp/bps)** — 0.01%。「涨 10 个基点」= 涨 0.10%。
- **久期 (duration)** — 对利率变化的敏感度。利润主要在很多年后的公司(典型如尚未盈利的成长股)= 长久期,对利率最敏感。
- **实际收益率 (real yield)** — 名义利率扣掉通胀预期。对成长股估值影响最直接。
- **收益率曲线 / 2s10s** — `10年利率 − 2年利率`。正常时长债 > 短债;**倒挂(负值)**历史上常被当衰退信号。
- **信用利差 (credit spread / HY spread)** — 高收益(垃圾)债比国债多要的利息。走阔=避险。
- **risk-on / risk-off** — 市场偏好风险 / 躲避风险。
- **金融环境 (financial conditions)** — 综合「钱松不松」,本项目用 NFCI 衡量。

**期权**
- **dealer(做市商)** — 期权交易的对手方,会对冲风险而被迫买卖正股。
- **Gamma(伽马)** — 衡量对冲量随股价变化的快慢的希腊字母。
- **GEX / net gamma exposure** — 全链净 gamma 敞口;正=抑制波动,负=放大波动。
- **gamma flip(翻转价)** — 净 gamma 由正转负的股价。
- **IV(隐含波动率)** — 期权价里隐含的未来波动预期;高=期权贵。
- **IV Rank / IV 分位** — 当前 IV 在历史区间里的相对高低(0–100%)。
- **OI(open interest,未平仓量)** — 某行权价上未平仓的合约数;免费数据滞后约 1 天。
- **cash-secured put(现金担保看跌)** — 卖出看跌期权并备好现金,被行权即在低位接货;IV 高时收的权利金更厚。
